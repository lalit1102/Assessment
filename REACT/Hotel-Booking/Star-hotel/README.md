# Assessment

## Star Hotel Booking App

This is a React application built using **Vite**, **Redux Toolkit**, and **Firebase**.

### Tech Stack
- React + Vite
- Redux Toolkit (RTK)
- Firebase Firestore
- React Router
- Bootstrap

### Features
- Room selection
- Book Now form
- Store booking data in Firebase
- Fetch bookings from Firebase
- Edit & Delete bookings (CRUD)

### Getting Started
```bash
npm install
npm run dev
